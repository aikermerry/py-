## 小整数对象池(常驻内存)

python 对小整数的定义是[-5,127]之间的这些数,这些数是提前建立好的.他们的地址都是一样的

```python
>>> a =100
>>> b =100
>>> id(a)
10935456
>>> id(b)
10935456


>>> a =1000
>>> b=1000
>>> id (a)
140708917449584
>>> id(b)
140708917449552
```
