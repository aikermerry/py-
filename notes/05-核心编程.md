# linux 多任务

操作系统运行中程序运行方式有并行与并发两种

并行:就是程序同时运行,但是大多是不可能实现的,电脑像现在四核的只能支撑四个

软件同时运行.只要程序一多就会有程序空闲,等待下次运行

并发:就是是软件在有限的电脑资源中轮流运行.看起来是所有软件都在运行,实际上是软件的切换速度非常快



操作系统的控制方式:

采用调度算法

​	优先级调用

​	时间片轮转



## python 中多任务执行

区分 进程,程序

**进程**:就是在运行的代码

**程序**:就是编写完毕的代码,没有运行

### fork

创建子进程

import os 

 ret  = os.fork()

在操作系统中父程序与子程序的执行先后顺序不一定 

os.getpid()获取当前程序的id

os.gitppid()获取当前成程序的父程序的id
